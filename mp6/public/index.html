Z, X, C: to select object

Key 0: to bind control to object
  • Arrows: Move the object
  • Shift+Arrows: Change the size of object
  • Control+Left/Right-Arrow: Rotate the object


Key 1: to bind controls to effect textures
  • Arrows: Move the selected effect texture
  • Shift+Arrow: Change the size of the effect texture
  • Control+Left/Right-Arrow: Rotate the effect texture
  • Keys: 3/4/5/6: Changes effect texture mode: Hide, Transparent, Override, Blend
  • Keys: N/M: Increase/Decrease the blend weight
